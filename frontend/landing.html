Self-Hosted приложение для заметок и управления знаниями. Markdown, вики-ссылки, графы, диаграммы, формулы — всё в одном месте.

Начать Возможности
Terminal
# Быстрый старт с Docker
$ mkdir izoldian && cd izoldian
$ curl -O https://raw.githubusercontent.com/roman-kvasnikov/izoldian/master/docker-compose.yml
$ docker compose up -d
# Откройте в браузере
$ open http://localhost:8000

Всё что нужно

Полноценная система управления знаниями на вашем сервере.

Markdown-редактор

Полная поддержка GFM с живым превью, split view, панелью форматирования и горячими клавишами.

Подсветка синтаксиса

30+ языков с нумерацией строк через highlight.js. Темы автоматически адаптируются.

Вики-ссылки и граф

Связывайте заметки через [[вики-ссылки]]. Визуализируйте граф знаний интерактивно.

Диаграммы Mermaid

Блок-схемы, sequence-диаграммы, диаграммы Ганта и другие — рендерятся прямо в заметках.

Формулы MathJax

Красивые LaTeX-формулы в строке и отдельным блоком. От простых выражений до сложных нотаций.

16+ тем оформления

Dracula, Nord, Catppuccin, One Dark, Rosé Pine, Solarized, GitHub и другие. Тёмные и светлые.

Многопользовательский + OIDC

Встроенная авторизация или подключение провайдера — Authelia, Keycloak и любой OIDC-совместимый сервис.

Публикация заметок

Делитесь заметками по публичным ссылкам с уникальными токенами. Одним кликом создать или отозвать доступ.

Мобильная версия

Адаптивный дизайн с нижней навигацией, удобными элементами управления и оптимизированной вёрсткой.
